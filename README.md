# Sistema de Agenda de Contatos

Sistema para o cadastramento de contatos em uma agenda, básico para treinamento e evolução das minha abilidades.

## Funcionalidades

- Cadastramento de contatos (nome, telefone, e-mail, categoria)
- Listar todos os contatos cadastrados
- Buscar contatos pelo nome, telefone ou e-mail
- Editar contatos já cadastrados 
- Remover contatos
- Limpar o formulário / cancelar a edição

## Como fazer funcionar na sua maquina/pc

1. Clona esse repositório
2. Acesse a pasta desse projeto
3. Execute o comando python main.py
